Weekend — Markt gesloten

De forex markt is gesloten in het weekend. EUR/USD en alle andere major pairs zijn niet te traden van vrijdagavond 22:00 tot zondagavond 22:00 (Belgische tijd).

Gebruik dit weekend om je week te evalueren, je strategie te herzien en je journal bij te werken.

Nog geen Client ID? Zo maak je er een aan →
Client ID aanmaken (5 min):
  1. Ga naar console.cloud.google.com en maak een nieuw project aan
  2. APIs & Services → Library → zoek "Google Drive API" → Activeer
  3. OAuth consent screen → External → vul naam in → voeg eigen Gmail toe als testgebruiker
  4. Credentials → Create Credentials → OAuth Client ID → Web application
  5. Bij "Authorized JavaScript origins" voeg je null toe
  6. Kopieer de Client ID en plak hem hierboven

Wat is een prop firm?
Een prop firm geeft je een gefinancierd account om mee te traden. Je mag winst houden (70-90%) maar er zijn strikte limieten: een max dagelijks verlies én een max totaal verlies. Overschrijd je één van beide → account verloren.

Prop firm regels uitgelegd
Max dagelijks verlies
Het maximale bedrag dat je op één dag mag verliezen. Bereik je dit plafond → je mag die dag niet meer traden of je verliest het account. Typisch 4-5% bij prop firms.
Max totaal verlies (drawdown)
Het totale verlies dat je account mag lijden t.o.v. de startwaarde. Typisch 8-10%. Vaak berekend op basis van hoogste balans (trailing drawdown) of beginbalans.
Gouden regel voor prop firms
Gebruik nooit meer dan 50% van je dagelijkse limiet als risico per dag. Als je max $500/dag mag verliezen → riskeer max $250/dag zodat je buffer hebt.

Regels om nooit te breken
  • ✕ Stop-loss verplaatsen richting verlies
  • ✕ Dagelijkse verliesilimiet prop firm overschrijden
  • ✕ Traden 30 min rond hoge impact news (NFP, CPI, ECB)
  • ✕ Meer dan 3 verliezende trades op één dag
  • ✕ Revenge traden na een verlies
